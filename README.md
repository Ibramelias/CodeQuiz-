# CodeQuiz
Simple Javascript quiz

## Running Test 
1. Open up the Quiz.
2. Select "Start Game" button.
3. Answer the questions.
4. As you answer them correctly or incorrectly the timer will be adjust.


## Build With
1. HTML
2. CSS
3. Bootstrap
4. Javascript
